# 🚀 Koleth Hook Master

!<img width="717" height="717" alt="indir" src="https://github.com/user-attachments/assets/5cddb9fa-3939-46e3-ad5a-47d4848dc6d8" />


**Koleth Hook Master**, Discord webhook'larını yönetmek, gelişmiş embed mesajları tasarlamak ve zamanlanmış mesajlar göndermek için geliştirilmiş, minimalist ve profesyonel bir masaüstü uygulamasıdır. Python ve CustomTkinter kullanılarak modern bir "Dark Mode" arayüzüyle tasarlanmıştır.

## ✨ Özellikler

*   **Gelişmiş Embed Editörü:** Başlık, Açıklama, Renk (Hex), Yazar (Author), Altbilgi (Footer), Resim ve Küçük Resim (Thumbnail) destekli tam teşekküllü embed tasarımı.
*   **Dosya Eki Desteği:** Sadece metin veya link değil, bilgisayarınızdaki herhangi bir dosyayı (resim, belge vb.) doğrudan Discord'a yükleyebilirsiniz.
*   **Zamanlayıcı (Scheduling):** Mesajlarınızı hemen göndermek yerine ileri bir saate (Örn: 14:30) zamanlayın. Uygulama arka planda bekler ve vakti gelince mesajı ateşler.
*   **Hızlı Profiller:** Sık kullandığınız webhook URL'lerini ve embed tasarımlarını profil olarak kaydedin, tek tıkla yükleyin veya silin.
*   **Güvenli Etiketleme (Mentions):** Yanlışlıkla sunucuyu rahatsız etmemek için `@everyone` ve `@here` etiketlerine izin verme/kapatma kontrolü.
*   **Canlı Terminal Logu:** Uygulama içindeki entegre terminal sayesinde gönderim durumlarını (Başarılı/Hatalı) anlık ve renkli olarak takip edin.
*   **Modern ve Büyütülebilir Arayüz:** CustomTkinter ile hazırlanmış, mat siyah ve Koleth moru vurgularına sahip, tam ekran yapılabilen akıcı tasarım.

## 🛠️ Kurulum ve Kullanım

### Seçenek 1: Kaynak Koddan Çalıştırma (Geliştiriciler İçin)

1.  Projeyi bilgisayarınıza indirin veya klonlayın:
    ```bash
    git clone https://github.com/Koleth/Koleth-Hook-Master.git
    cd Koleth-Hook-Master
    ```
2.  Gerekli kütüphaneleri yükleyin:
    ```bash
    pip install -r requirements.txt
    ```
3.  Uygulamayı başlatın:
    ```bash
    python main.py
    ```

### Seçenek 2: .exe Olarak Kullanma (Son Kullanıcılar İçin)

Eğer Python kurmakla uğraşmak istemiyorsanız, [Releases](#) (Sürümler) sekmesinden uygulamanın derlenmiş `.exe` versiyonunu indirip doğrudan çalıştırabilirsiniz.

*(Not: .exe versiyonunda profilleriniz güvenli bir şekilde `%APPDATA%\KolethHookMaster\profiles.json` konumuna kaydedilir, böylece programı güncelleseniz bile verileriniz kaybolmaz.)*
